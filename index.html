自動送信 設定手順

1LINE公式アカウント — Messaging API有効化

  1. manager.line.biz にログイン
  2. 対象アカウントを選択 →「設定」→「Messaging API」
  3. 「Messaging APIを利用する」をタップ → プロバイダーを作成
  4. 有効化後、「チャネルアクセストークン（長期）」を発行してコピー
⚠️ トークンは長い文字列です。メモ帳に保存しておいてください。

2お父さんのユーザーIDを取得

  1. お父さんにLINE公式アカウントを友だち追加してもらう
  2. LINE Developersコンソール（developers.line.biz）にアクセス
  3. チャネル →「Messaging API」→「Webhook設定」欄の下にある「友達」リストからUserIDを確認
  4. または：お父さんがLINE公式アカウントにメッセージを送ると、Webhookログにユーザーが表示されます
💡 簡単な方法：LINEで「テスト」とメッセージを送ってもらいログを確認

3Googleスプレッドシート + GASを作る

  1. Googleスプレッドシートを新規作成（名前：渡部造園_現場管理）
  2. 「拡張機能」→「Apps Script」を開く
  3. 下のコードを全部コピーしてApps Scriptに貼り付け
  4. YOUR_TOKENとYOUR_USER_IDを自分のものに書き換える
  5. 「デプロイ」→「新しいデプロイ」→ 種類：ウェブアプリ → アクセス：全員 → デプロイ
  6. 表示されたURLを⚙️設定タブに貼り付ける
// 渡部造園 現場情報共有 - Google Apps Script // ★ 以下2行を自分の値に書き換えてください const LINE_TOKEN = 'YOUR_CHANNEL_ACCESS_TOKEN'; const LINE_USER_ID = 'YOUR_FATHER_USER_ID'; const SHEET_NAME = '現場情報'; function doPost(e) { try { const data = JSON.parse(e.postData.contents); // スプレッドシートに記録 const ss = SpreadsheetApp.getActiveSpreadsheet(); let sheet = ss.getSheetByName(SHEET_NAME); if (!sheet) { sheet = ss.insertSheet(SHEET_NAME); sheet.appendRow(['送信日時','ステータス','顧客名','現場住所','工事内容', '見積日時','作業開始','作業終了','集合時間','担当者','見積金額','備考']); } sheet.appendRow([ new Date().toLocaleString('ja-JP'), data.status, data.client, data.address, data.work, data.estimateDate, data.startDate, data.endDate, data.meettime, data.staff, data.price, data.memo ]); // LINEメッセージ送信 const msg = buildMessage(data); sendLine(msg); return ContentService .createTextOutput(JSON.stringify({ok:true})) .setMimeType(ContentService.MimeType.JSON); } catch(err) { return ContentService .createTextOutput(JSON.stringify({ok:false, error:err.message})) .setMimeType(ContentService.MimeType.JSON); } } function buildMessage(d) { const fmt = s => s || '—'; return [ '━━━━━━━━━━━━━━', '【現場情報】渡部造園', '━━━━━━━━━━━━━━', `▶ 状態　　：${fmt(d.status)}`, `▶ 顧客名　：${fmt(d.client)}`, `▶ 現場　　：${fmt(d.address)}`, `▶ 工事　　：${fmt(d.work)}`, '──────────────', `▶ 見積日時：${fmt(d.estimateDate)}`, `▶ 作業開始：${fmt(d.startDate)}`, `▶ 作業終了：${fmt(d.endDate)}`, `▶ 集合　　：${fmt(d.meettime)}`, '──────────────', `▶ 担当　　：${fmt(d.staff)}`, `▶ 見積金額：${fmt(d.price)}`, '──────────────', '▶ 備考', fmt(d.memo), '━━━━━━━━━━━━━━', ].join('\n'); } function sendLine(message) { UrlFetchApp.fetch('https://api.line.me/v2/bot/message/push', { method: 'post', headers: { 'Content-Type': 'application/json', 'Authorization': 'Bearer ' + LINE_TOKEN }, payload: JSON.stringify({ to: LINE_USER_ID, messages: [{ type: 'text', text: message }] }) }); } // 接続テスト用 function doGet(e) { return ContentService .createTextOutput(JSON.stringify({ok:true, msg:'GAS is running'})) .setMimeType(ContentService.MimeType.JSON); }

4⚙️ 設定タブに入力して完了

  1. ⚙️設定タブを開く
  2. GAS Web App URLを貼り付け
  3. お父さんのLINEユーザーIDを入力
  4. 「接続テスト」→ ✅ が出れば完了！
✅ 設定完了後は入力タブから「LINEに自動送信」ボタン1つで送れます。
